Getting 403 Forbidden from Ollama?

Ollama blocks requests from browser extensions by default. Allow them by setting the OLLAMA_ORIGINS environment variable and restarting Ollama:

Windows (then restart Ollama):
setx OLLAMA_ORIGINS "chrome-extension://*"

macOS / Linux:
export OLLAMA_ORIGINS="chrome-extension://*"
How to install Ollama

Ollama runs LLMs locally on your machine — no data leaves your computer.

  1. Download the installer from ollama.com/download (Windows, macOS or Linux) and run it.
  2. Open a terminal window:
    • Windows: press Win, type cmd, press Enter
    • macOS: press Cmd + Space, type Terminal, press Enter
  3. Copy the command below, paste it into the terminal and press Enter. It downloads the translation model (a few GB — wait until it shows success):
ollama pull gemma4:e4b

4. Verify it is running — open http://localhost:11434 in the browser (it should say "Ollama is running"), then click Connect above to load your models.
